## 题目描述
月影斑驳，泛舟独饮。

粼粼水波，令你遐想连篇，星河璀璨，瑰丽变幻，你的眼前突然展现出绚烂的景象，一切的光影舞动着，交织着，梦境般的宏伟壮阔联结过去与未来，金黄的火光环绕世界。潋滟江水也明朗起来，奔涌起来，那一叶扁舟忽地幻化为万点星火，旋转着向月而去，你的身体卷云般变得飘渺，亦缓缓而起，你闭上清澈的双眼，解构世界的梦浮出水面，又一瞬则坠入虚无，意义不复存在……
“我该在哪里停留？我问我自己。”你不将停留，意义的本质即是运动，你将走向无穷的变化，生生不息。

你注视着眼前的水流，潺潺淌过曲折的河道，悦耳空灵的水声，撒满水面的万家灯火，你意识到，电流不就像那水流一样吗？  

你抬头仰望，天空中隐隐地还有云彩染着晚霞的余晖，飘荡在夜幕之中，地面上的灯火，也不知不觉地化作繁华的城市，霓虹灯与车流划过的亮带在一切空间中交错，映出大厦玻璃幕墙魔幻的光影，你不禁思考，如此庞大的城市，它供电系统究竟如何运转呢？经过一番探索，你发现整个供电网络将每一户之间相互连接，最终形成了一棵树的结构。城市中的每家每户都有三根可以当作零电阻的导线，如果这一家和另一家在电网树中有连边，那么这一家的三根导线会和另一家的三根导线全部由电网中的电线相连，总共形成九条电线。这九条电线可以视作九个的电阻，也就是说，每条电线都有其阻值。至于仅仅与一户在电网上有连接的非第一户的住户（也就是叶子）的三根导线的另一端会全部通过电网中的无电阻的电线连到发电厂去，发电厂可以视作电池，它将再在电网中延伸出三根无电阻的电线通往第一户的三根导线。因此，我们假定第一户为“树根”。电流的流动方向便是从树根流向叶子。从叶子到发电厂以及从发电厂到根的电路部分你可以不做考虑。

当然，在这个城市极长的时间跨度下，你发现连接不同户的导线的电线的电阻会有变化。在某一些时间点，你想要知道某一户的某一个导线和另一户的某一个导线之间的等效电阻是多少。
## 输入格式
{{s.input_file()}}

第一行包含一个正整数 $n$，表示住户的个数。

接下来 $4(n-1)$ 行，每 $4$ 行为一个分段。

第 $i$ 个分段的第一行为一个整数 $f_{i+1}$，表示编号为 $i+1$ 的住户的父节点为 $f_{i+1}$。

接下来一个 $3\times3$ 的矩阵 $g_{xy}$，第 $x$ 行第 $y$ 个数表示编号为 $i+1$ 的插线板的第 $x$ 个导线和 $f_{i+1}$ 的第 $y$ 个导线之间的**电导** $G$ （$G=\frac{1}{R}$）。 

接下来一个整数 $q$，表示记录了 $q$ 个时间点。

第一个整数 $k$，若 $k=1$ 则接下来包含四个整数 $a_i,x_i,x_j,G$，表示将 $a_i$ 户的第 $x_i$ 个导线与 $a_i$ 户的父节点 $f_{a_i}$ 的第 $x_j$ 个导线之间的**电导**值改为 $G$。保证 $2\le a_i\le n$。

若 $k=2$，则接下来包含四个整数 $a_i,x_i,a_j,x_j$，表示询问第 $a_i$ 户的第 $x_i$ 个导线与第 $a_j$ 户的第 $x_j$ 个导线之间的**电阻**大小。保证 $a_i\neq a_j$。
## 输出格式
{{s.output_file()}}
对每个询问，输出一行实数，表示两条线之间的电阻。 

设选手输出为 $x$，标准输出为 $y$，若$|\frac{x-y}{y}|\le 10^{-3}$，则被认为该答案正确。

## 数据范围
对于所有测试数据，$0<G\le 10$。 


| 测试点编号      | $n\le$   | $q\le$   | 特殊性质 |
| ---------- | -------- | -------- | ---- |
| $1\sim6$   | $100$    | $1,000$  | A    |
| $7\sim10$  | $1,000$  | ^        | B    |
| $11\sim16$ | $10,000$ | $10,000$ | C    |
| $17\sim20$ | ^        | ^        | 无    |

特殊性质 A：保证输入中只有查询没有修改。

特殊性质 B：保证数据是一条链。

特殊性质 C：保证树高不超过 $30$。
